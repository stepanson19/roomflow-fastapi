# Варианты тем

1. RoomFlow - бронирование репетиционных комнат с подбором свободного слота по бюджету, оборудованию и вместимости.
2. PrintQueue - лаборатория 3D-печати с очередью заказов и расчетом оптимального порядка печати.
3. FreshDesk Cafe - управление столами антикафе с расчетом загрузки, бронированием и подбором места под компанию.
4. RestoreLab - очередь реставрационных работ в музее с приоритизацией экспонатов по состоянию и дедлайну.

Выбран вариант RoomFlow, потому что его легко показать через Swagger, в нем естественно появляются связи между таблицами, роли пользователей и понятная алгоритмическая задача.
